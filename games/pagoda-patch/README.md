# Pagoda Patch · ခြံစောင့်

Burmese-flavored Plants vs Zombies — part of **YGN Arcade**.

## Play

From the monorepo root:

```bash
npx serve .
# open /games/pagoda-patch/
```

Or open this folder’s `index.html` via any static server.

## Features

- 8 campaign levels with unique twists + endless mode
- Procedural Burmese-inspired music & SFX
- Star ratings saved in `localStorage`

## Controls

- Click sun · seed · lawn tile  
- `1–9` plants · `S` shovel · `Esc` pause
